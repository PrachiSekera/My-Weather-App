# Weather Dashboard

A modern and responsive weather application built using HTML, CSS, JavaScript, Bootstrap, and the OpenWeather API. The application provides real-time weather information, dynamic weather-based visuals, dark/light mode support, search history management, and location-based weather detection.

## Features

### Weather Information
- Real-time weather data using OpenWeather API
- Search weather by city name
- Current temperature display
- Feels-like temperature
- Humidity information
- Pressure information
- Visibility information
- Minimum and maximum temperature

### Dynamic User Experience
- Weather-based animated backgrounds
- Sunny weather animation
- Cloud animation
- Rain animation
- Snow animation
- Thunderstorm effects

### User Convenience
- Automatic location-based weather detection using Geolocation API
- Search history with Local Storage
- Dark Mode and Light Mode support
- Persistent theme preferences
- Enter key support for quick searching

### Additional Features
- Multi-city weather comparison table
- Responsive design for desktop, tablet, and mobile devices
- Glassmorphism-inspired UI
- Smooth transitions and hover effects

## Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/weather-dashboard.git
```

2. Navigate to the project directory

```bash
cd weather-dashboard
```

3. Open `index.html` in your browser.

## API Setup

This project uses the OpenWeather API.

1. Create an account at OpenWeather.
2. Generate an API key.
3. Replace the API key inside `script.js`.

```javascript
const apiKey = "YOUR_API_KEY";
```
